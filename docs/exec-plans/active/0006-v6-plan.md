## 목표

v6 = **P5 Following MVP** — 아티스트 팔로우/언팔로우 기능.

v5에서 P4 인증이 완료되었고, v3/v4에서 Artist 엔티티(P2)가 완료되었다. P5의 두 전제가 모두 충족됐으므로 v6에서 Following을 착수한다.

### 왜 P5 Following인가

- **전제 충족**: P2(Artist 엔티티) + P4(인증) — 둘 다 완료
- **비전 임계 경로**: ②(Following) → ③(디깅/커뮤니티) → ⑦(추천/알림) 전체의 출발점
- **P6 My Record 대비**: P6는 V-5(관람 기록 방식) 사람 결정이 미확정. P5는 미확정 결정 없음

**비범위(v6 밖):** 팔로우 기반 추천/알림(P7), My Record(P6 — V-5 결정 후), 가격대 목록 필터.
