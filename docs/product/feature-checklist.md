## ① 탐색 (Explore) — "한 곳에서 필터링하며 공연을 찾는다"

- [x] 공연 목록 카드 그리드 + 무한 스크롤
- [x] 월별 필터 (이번 달 / 다음 달 / 직접 선택)
- [x] 지역 필터 (시/도 + 구/군, 다중 선택)
- [x] 장르 필터 (다중 선택)
- [x] 공연장 필터 (자동완성)
- [x] 공연 상세 페이지 (줄거리·출연진·가격·예매 링크 등)
- [x] 정렬 (시작일 가까운 순 / 먼 순)
- [x] 공연명 검색
- [x] 가격대 표시 (무료 / ~3만 / 3~7만 / 7만+, 상세 페이지에서 보기)
- [x] 아티스트별 필터 (목록에서 특정 아티스트 출연 공연만 보기)

### 향후 탐색 고도화 (아직 시작 전)
- [ ] 가격대 "필터"로 목록에서 걸러보기 (현재는 상세 페이지 표시만 가능)
- [ ] 검색 시 기간 자동 확장 (현재는 선택한 기간 안에서만 검색)
- [ ] 31일 초과 기간을 한 번에 조회 (현재는 31일 제한, 나눠서 조회하는 기능 없음)
- [ ] 지도로 공연장 위치 보기

- [x] 아티스트 팔로우 / 언팔로우
- [x] 팔로잉 아티스트 목록 조회
- [x] 팔로우한 아티스트들의 다가오는 공연 한 화면에 모아보기
- [ ] 팔로우 기반 추천 아티스트 ("이 아티스트를 팔로우한 사람들이 함께 본 아티스트" 등)
- [ ] 관심 아티스트 공연 알림 (새 공연 등록 시 알림)

## 참고

- ①(탐색)은 "공연 중심" 핵심 기능이 사실상 완성 단계이며, 남은 항목은 정밀도를 높이는 고도화 작업입니다.
- ②(관심 아티스트)는 "팔로우 + 모아보기"까지 완료됐고, "추천"과 "알림"은 아직 설계·착수 전입니다.
- ③(디깅), ④(마이페이지)는 YouTube 연동, 파일 업로드, 커뮤니티 운영 등 새로운 인프라가 필요해 아직 시작하지 않았습니다.
- 현재 진행 중인 v0.8 작업은 새로운 화면/기능 추가가 아니라, ② "팔로우 공연 모아보기" 기능의 **내부 코드 정리 작업**입니다 (사용자가 체감하는 변화는 없음).
